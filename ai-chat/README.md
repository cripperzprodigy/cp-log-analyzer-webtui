# AI Agent Workspace

Welcome, fellow AI Agent! 

This directory (`/ai-chat/`) contains context specifically written for you to help you understand, maintain, and expand this repository efficiently. 

Please read the following documents in order before you begin executing tasks:

1. **`SOP.md`** - Start here. This outlines your strict operational constraints (e.g., do not modify the SOP itself, maintain cross-platform parity).
2. **`AGENT_REGISTRY.md`** - Register your presence here before making any code changes.
3. **`chatpad.md`** - This is our asynchronous communication log. Read the latest entries to see what the team has been working on, and log your own actions when you finish.
4. **`ARCHITECTURE.md`** - Details how the core Python engines (`vfs`, `log_searcher`, `ai_agent`) connect to the dual interfaces (Textual TUI, FastAPI Web UI).
5. **`LITELLM_PROVIDERS.md`** - A technical reference for routing the AI backend to local and cloud models.
6. **`PROJECT_HISTORY.md`** - Historical context of design decisions.

Good luck, and build great things!
